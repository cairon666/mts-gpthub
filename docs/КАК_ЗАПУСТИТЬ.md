# Как запустить dev окружение

## Зависимости

Для запуска необходимо установить: 

- утилита [uv](https://docs.astral.sh/uv/getting-started/installation/)
- python >= 11 
- [nodejs](https://nodejs.org/en/download) <= 22.22.2. Установить советую через nvm или fnm для POSIX. Для Windows через бинарник `.msi`
- [docker](https://docs.docker.com/engine/install/)

## backend

 ```bash
uv sync # усановка зависимостей из uv.lock

# запуск backend
# первый запуск может быть не быстрым и не сразу отобразиться на фронте, необходимо подождать пока загрузятся необходимые модули
uv run uvicorn open_webui.main:app --reload --host 0.0.0.0 --port 8080

# добавить пакет
uv add <имя_пакета> 
uv add "fastapi==0.115.0" # конкретную версию

# если скачали новую зависимость, то нужно ее добавить в uv.lock тоже, чтобы другие члены команды ее могли тоже подтянуть
uv lock
```

## frontend

```bash
# установка зависимостей
npm i 

# запуск приложения
# каждый запуск подтягивает wasm python модули, поэтому не быстро
# после запуска в консоль отпишет ссылке, по которой фронт будет доступен. по умолчанию всегда http://localhost:5173
npm run dev
```
